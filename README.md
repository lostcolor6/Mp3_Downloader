# Mp3_Downloader
Simple Application with GUI (PyQt5) to download Music from Youtube as .mp3 files
You can choose to insert 
- one link
- a playlist link



### Requirements
````
pip install yt-dlp
pip install PyQt5 
```` 

also make sure `ffmpeg` is installed (still works without but will throw errors)
